
# 🍽️ Flavio Restaurant Website

A fully responsive multi-page restaurant website built using HTML, CSS, and JavaScript.  
This project is designed to provide a modern and elegant restaurant experience with interactive UI sections, dynamic menu features, reservation forms, and responsive layouts.

## 🚀 Live Demo
https://arjunmeena001.github.io/Restaurant-website/

---

## 📌 Features

- Fully Responsive Design
- Multi-Page Website
- Modern Restaurant UI
- Interactive Navigation Bar
- Dynamic Food Menu Toggle
- Add To Cart UI Design
- Reservation Booking Form
- Restaurant History Timeline
- Contact Page with Google Map
- Animated Sections & Hover Effects
- Social Media Icons
- Mobile Friendly Layout
